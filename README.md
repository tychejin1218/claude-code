## 적용 방법

각 프로젝트 디렉토리의 `.claude/`, `CLAUDE.md`, `.claudeignore`를 신규 프로젝트에 복사하여 사용

```bash
cp -r {프로젝트}/.claude/ /path/to/your-project/
cp {프로젝트}/CLAUDE.md /path/to/your-project/
cp {프로젝트}/.claudeignore /path/to/your-project/
```
